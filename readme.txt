== Description ==

WordPress图片同步到腾讯万象优图，用户访问时加载腾讯万象优图服务器上图片，有效提高WordPress站点访问速度，减轻服务器压力。

== Installation ==

1.下载插件
2.上传到/wp-contents/plugins/目录
3.启用插件
