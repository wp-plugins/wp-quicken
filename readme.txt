通过禁用Google字体、emoji表情等使WordPress速度更快，更符合国人使用。

== Description ==

这款插件主要是针对国人需求开发的，WordPress本来就很蜗牛了，默认还要加载一系列海外资源，使得我们的博客更蜗牛，安装这款WordPress加速优化插件WP Quicken可以帮助大家一键优化加速。

== Installation ==

1.下载插件
2.上传到/wp-contents/plugins/目录
3.启用插件
